## ✏️ LP編集方法（クライアント様向け）

技術知識がなくてもブラウザだけで完結する設計です：

1. **AI（Claude.ai無料版）に相談**
   - 「○○を△△に変えたい」と日本語で依頼
2. **GitHub Web Editor で編集**
   - https://github.com/koki-187/scalp-ink-lp → 該当ファイル → ✏️ Edit
   - `Ctrl + F` で検索 → コードを貼り付け
3. **Preview で差分確認** → **Commit changes**
4. **1〜2分で本番反映**（Cloudflare Pages 自動デプロイ）

詳細は `クライアント納品_2026-04-23/無料プランLP編集_超詳細ガイド.pdf` を参照。

## 🚀 デプロイ

`master` ブランチへの push を Cloudflare Pages が自動検知し、約1〜2分でビルド＆公開されます。
ローカル環境やCI/CDパイプラインは不要です。

```bash
git pull origin master
# 編集
git add <file>
git commit -m "変更内容を日本語で簡潔に"
git push origin master
# → 1〜2分後に cima-scalpink.com に反映
```
