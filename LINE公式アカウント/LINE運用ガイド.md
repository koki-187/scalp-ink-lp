## 1. アカウントの目的

CIMA SCALP INKのLINE公式アカウント（@045ovoxr）は、スカルプマイクロピグメンテーション（SMP）サービスの認知拡大と予約促進、既存顧客との継続的な関係構築を目的とする。

### 運用の3つの柱

1. **新規顧客の獲得**
   施術事例やビフォーアフターの配信で興味喚起し、予約・相談への導線を確保する

2. **顧客ロイヤルティの向上**
   施術後のアフターケア情報やメンテナンス時期のリマインドで再来店率を高める

3. **ブランド信頼性の構築**
   専門知識の発信、安全性の訴求、実績紹介で沖縄No.1 SMPサロンとしての地位を確立する

## 3. 自動応答メッセージ運用

### 友だち追加時のあいさつメッセージ

2つの吹き出しで構成。第1吹き出しでサロン紹介、第2吹き出しで予約方法を案内。
- `{友だちの表示名}` と `{アカウント名}` の差し込み変数を使用
- サロンの4つの強み（韓国発技術・世界大会入賞・明朗会計・体験施術価格）を明記
- リッチメニューと電話の両方に誘導

### トークで予約テンプレート

予約時に必要な情報（名前・希望日時・メニュー・気になる箇所）を箇条書きで案内。
- メニューと参考価格を併記し、ユーザーが選びやすく
- 第2希望まで聞くことで調整の手間を削減

### お問い合わせ受付テンプレート

問い合わせの敷居を下げるカジュアルなトーン。
- 「気軽に話しかけてください」で心理的ハードルを低減
- プライバシー保護の明示（他の客には公開されない）

### 営業時間外自動応答

営業時間（月〜土 10:00-19:00）外に受信したメッセージに自動返信。
- 翌営業日対応の案内
- お急ぎの場合はWeb予約・電話に誘導

### キーワード応答

| キーワード | 応答内容 |
|-----------|---------|
| 料金 / 値段 / 価格 / プラン | 料金表・メニュー一覧（1ドット=1円、体験¥3,000〜） |
| 場所 / アクセス | 〒901-2223 沖縄県宜野湾市野嵩2-34-3 + Google Maps + 営業時間 |
| 予約 / 相談 / カウンセリング | 予約方法3つ（トーク・Web・電話）+ 完全予約制の案内 |

## 5. 配信ルール

### 基本設定

| 項目 | 設定値 |
|------|--------|
| 配信タイミング | 平日の夕方 18:00 |
| 配信頻度 | 週3回（月・水・金） |
| 1メッセージの文字数 | 500文字以内 |

### 配信時の注意事項

1. **ブランド統一:** 画像は高品質、黒×ゴールドのブランドイメージを維持する
2. **CTA必須:** 必ずCTAボタンを設置する（予約ページ・電話・Instagram）
3. **即時対応:** 配信後24時間以内のチャット返信を徹底する
4. **トーン:** 温かく専門的。傷跡・薄毛の方には特に丁寧で希望を持てるメッセージを心がける
5. **プライバシー:** ビフォーアフター写真は必ず掲載同意書を取得してから使用する
